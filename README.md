# Техническое задание к проекту “Курс по линейной алгебре: Матрицы”

1.	## Общая характеристика проекта

Проект включает в себя целостное приложение с многостраничной структурой. Проект включает в себя несколько модулей:
- Методическая часть 
- Калькулятор для вычислений
- Тестовая часть
- Раздел для отзывов пользователей

2.	## Описание интерфейса

**Главное окно:**

- Заголовок с названием курса.
- Фотография с темой курса
- Информационный блок с текстом
- Список тем (что такое матрицы, транспонирование, операции с матрицами, метод крамера), где пользователь выбирает интересующую тему
- Кнопка “Перейти”. При нажатии открывается окно с методической частью
- На главном окне должна быть кнопка "Отзывы": открывает окно для написания и редактирования отзыва. 

**Методическая часть:**

- Заголовок методической темы
- Информационный блок с текстом и фотографиями
- В правом верхнем углу должна быть кнопка “Скачать материал”. По нажатию скачивается теоретический материал темы в .txt формате
- В темах “транспонирование”, “операции с матрицами”, “метод Крамера” должны быть калькуляторы. В первом случае калькулятор позволяющий производить транспонирование матриц. В теме “Операции матриц” должен быть калькулятор позволяющий складывать, вычитать, умножать. В теме “метод Крамера” калькулятор должен считать матрицы методом Крамера и выдавать ответ.
- Калькулятор должен содержать кнопки: "Рассчитать".
- Каждый калькулятор должен принимать файл с матрицами .txt.
- В конце каждого документа должна быть кнопка “Начать тест”. После нажатия будет открываться тест с радиокнопками.

**Тестовая часть:**

- Должен быть текст вопроса
- Радиокнопка с вариантом ответа
- Кнопка “завершить тест”
- В конце должен показываться результат теста (Вы набрали n  зи m баллов)

**Раздел для отзывов пользователей:**
- Текстовое поле для редактирования, форматирование (жирный, курсив).
- Кнопка "Отправить".

3.	## Описание баздынных

Должна быть база данных:
- Результатов (айди теста, имя пользователя, айди темы, количество баллов, дата проходения)
- Тема (айди темы, название темы)
- Отзыв (айди отзыва, ник пользователя, название темы отзыва, отзыв, дата )

4.	## Функции калькулятора
- Ввод данных вручную или из файла.
- Выполнение вычислений (прописанно в описании интрфейса)
- Вывод результата

5.	## Тест
- Длина теста 10 вопросов.
- Тип вопросов Радиокнопки 
- После завершения теста показывается количество правильных ответов
- Отображаются правильные ответы для ошибок.

6.	## Требования 

- Цветовая схема должна быть спокойные, нейтральные тона.
- Интерактивные элементы (кнопки, вкладки) выделяются цветом.
- Удобная навигация между разделами.
- Весь код хранится на GitHub с README, описывающим проект и инструкциями.

7.	## Перечень основных функций
- Открытие существующего файла (.txt).
- Скачивание файла.
- Ввод, удаление и редактирование текста, отмена и повтор, копировать, вырезать, вставить, выделить всё.
- Форматирование текста: шрифт, размер, стиль, цвет, выравнивание текста.
- Поиск и замена текста.
- Экспорт в .txt.
